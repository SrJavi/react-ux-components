# Requirements

React Components UX requires the following be installed in your project:

- React 17.0.2+

# Installation

### npm

```sh
npm install react-components-ux
```

### yarn

```sh
yarn add react-components-ux
```

## Start Development Environment

1. Clone this repository;<br />
2. Run `npm or yarn install` at each project in order to install dependencies.<br />
3. Run `yarn storybook`.<br />
4. Access `localhost:8080` in your browser.<br />
5. Run `yarn build` to generate your own build.<br />
